# Student Management System (Java)

This project is a simple Student Management System built using Java and basic web technologies.

## Features
- Add student details
- View student list
- Update student information
- Delete student records
- Uses Object-Oriented Programming concepts

## Technologies Used
- Java (Core Java)
- OOP Concepts (Abstraction, Inheritance, Interface)
- HTML
- CSS
- JavaScript

## Project Structure
- main.java → Java backend logic
- index.html → UI structure
- style.css → Styling
- script.js → Frontend logic

## How to Run
1. Clone or download this repository
2. Open the project in VS Code or any IDE
3. Run `main.java` to test backend logic
4. Open `index.html` in a browser to see frontend UI

## Purpose
This project is created for learning and interview preparation.
